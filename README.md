# Исследование влияния джиттера и потерь пакетов на качество потокового видео (QoE)

**Автор:** Селиверстова Алиса

## О проекте
Код для моей дипломной работы. Программа эмулирует потери пакетов и джиттер в IP-сети и оценивает влияние этих параметров на качество видео с помощью метрики VMAF.

## Файлы
- `sender_final.py` — вносит искажения в видеопоток и отправляет его
- `receiver_final.py` — принимает искажённый поток и сохраняет видео
- `run_final_no_vmaf.py` — автоматически прогоняет 216 тестов
- `calculate_vmaf.py` — рассчитывает метрику VMAF
- `plot_graphs.py` — строит графики зависимостей
